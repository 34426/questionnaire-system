### 管理员功能:

登录、个人中心、管理员管理、用户管理、问卷管理、题目管理、问卷调查记录、新闻资讯管理、轮播图管理

### 用户功能:

注册、登录、首页、问卷列表、进行问卷调查、查看问卷调查记录、新闻资讯、个人中心
